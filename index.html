Personal Project

Ultimate Pokedex

Built with React, HTML5, SASS/CSS, NodeJS, Express, Massive, PostgreSQL, and BCrypt, this project was created to have data from all the Pokemon games in one easily-accessible place. All users can navigate through the different regions to find data on all Pokemon, items, moves, and types from the Pokemon games. Users that create an account can customize their own team by adding up to six Pokemon and giving them nicknames. The user's profile picture can also be customized. Mobile users will find the application to be fully responsive with no part of the experience lost to the smaller screen size.
