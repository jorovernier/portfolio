Projects

BudBud

GitHub Repo

Budget Buddy is an all-in-one financial assistance application. With BudBud, users can track their income and expenses with responsive charts visually displaying information added to their account. Users can also add bank accounts and credit cards to keep track of their savings and card usages. This project is still a work in progress. Built with React, JavaScript, Node, Express, PostgreSQL, Bootstrap, CSS, and ChartJS.

Credit Congregate

GitHub Repo

Credit Congregate is a one-stop shop for looking at the many credit cards available from banking institutions nation-wide, filtering them based on certain criteria, adding them to a want list, and recording all of the cards you currently have. This project is still a work in progress. Built with JavaScript, HTML5, CSS3, Node, Express, and PostgreSQL.

ACNH Catch Tracker

GitHub Repo

The Animal Crossing Catch Tracker was commissioned while working at Vern Software Development. Built with React, JavaScript, SASS, CSS, Node, Express, Massive, PostgreSQL, and BCrypt, this application was created to track bugs and fish caught in Animal Crossing New Horizons. Additionally, users can also keep track of how many flowers they've grown and what fossils they've donated.

Ultimate Pokedex

GitHub Repo

Built with React, JavaScript, SASS, CSS, ChartJS, Node, Express, Massive, PostgreSQL, and BCrypt, this application was professionally created to have data from all the Pokemon games in one easily-accessible place. All users can navigate through the different regions to find data on all Pokemon, items, moves, and types from the Pokemon games. Users that create an account can customize their own team by adding up to six Pokemon and giving them nicknames. The user's profile picture can also be customized. Mobile users will find the application to be fully responsive with no part of the experience lost to the smaller screen size.

Void

GitHub Repo

Created in a group with: Gina Perez, Thomas Hoover, and Isaac Hansen. Built with React, JavaScript, SASS, CSS, ChartJS, Node, Express, Massive, PostgreSQL, BCrypt, React Drag-and-Drop, and Socket.io, this application was created to mix popular communication platform Discord and task tracking site Trello together. Users can create a group, add other users, columns, and tasks to the group, and they can change the name of their groups, columns, and tasks. Tasks can be moved by dragging them to the desired column. Users can also change their username, email, password, and profile picture through their Profile page. Mobile users will find the site to be fully responsive.
