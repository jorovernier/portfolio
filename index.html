Projects

Ultimate Pokedex

Website • GitHub Repo

Built with JavaScript, React, HTML5, SASS/CSS, ChartJS, NodeJS, Express, Massive, SQL, PostgreSQL, and BCrypt, this project was professionally created to have data from all the Pokemon games in one easily-accessible place. All users can navigate through the different regions to find data on all Pokemon, items, moves, and types from the Pokemon games. Users that create an account can customize their own team by adding up to six Pokemon and giving them nicknames. The user's profile picture can also be customized. Mobile users will find the application to be fully responsive with no part of the experience lost to the smaller screen size.

Void

Website • GitHub Repo

Created in a group with: Gina Perez, Thomas Hoover, and Isaac Hansen. Built with JavaScript, React, HTML5, SASS/CSS, ChartJS, NodeJS, Express, Massive, PostgreSQL, BCrypt, React Drag-and-Drop, and Socket.io, this project was professionally created to mix popular communication platform Discord and task tracking site Trello together. Users can create a group, add other users, columns, and tasks to the group, and they can change the name of their groups, columns, and tasks. Tasks can be moved by dragging them to the desired column. Users can also change their username, email, password, and profile picture through their Profile page. Mobile users will find the site to be fully responsive.

ACNH Catch Tracker

Website • GitHub Repo

Built with React, HTML5, SASS/CSS, NodeJS, Express, Massive, PostgreSQL, and BCrypt, this project was created to track bugs and fish caught in Animal Crossing New Horizons. Additionally, users can also keep track of how many flowers they've grown and what fossils they've donated. This is a side project done for fun!
